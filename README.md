# 認証コード通知 (TOTP Notification Extension)

Google Authenticatorのような2段階認証（TOTP）コードをブラウザ上で生成し、デスクトップ通知で受け取ることができるChrome拡張機能です。

## ✨ 主な機能
- **TOTPコード生成**: 登録したアカウントの6桁の認証コードをリアルタイムに表示。
- **デスクトップ通知**: 設定した間隔で認証コードをプッシュ通知。
- **ダークモード対応**: 目に優しいダークテーマに切り替え可能。
- **ローカル保存**: シークレットキーはブラウザ内のセキュアなストレージにのみ保存されます。

## 🛠 インストール方法（開発者モード）

この拡張機能はパッケージ化される前の「ソースコード」の状態です。以下の手順でChromeに導入してください。

1. **ファイルを準備する**
   - このリポジトリをダウンロード（ZIPで保存して解凍、または `git clone`）し、一つのフォルダにまとめておきます。
   - フォルダ内には以下のファイルが含まれていることを確認してください：
     - `manifest.json`
     - `popup.html`
     - `popup.js`
     - `background.js`

2. **Chromeの拡張機能管理画面を開く**
   - ブラウザのURLバーに `chrome://extensions/` と入力して開くか、メニューの「設定」＞「拡張機能」を選択します。

3. **デベロッパーモードをオンにする**
   - 画面右上の **「デベロッパー モード」** のスイッチをオンにします。

4. **フォルダを読み込む**
   - 画面左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
   - 手順1で用意した、ファイルが入っている**フォルダまるごと**を選択します。

5. **完了**
   - ブラウザのツールバーにあるパズルのピース型アイコン（拡張機能メニュー）から「認証コード通知」を選択し、ピン留めすると便利です。

## 📖 使い方
1. 拡張機能のアイコンをクリックしてポップアップを開きます。
2. 「アカウント名（例: Google）」と「シークレットキー（Base32形式）」を入力して追加します。
3. 表示されたコードをタップするとクリップボードにコピーされます。
4. 「通知を有効化」にチェックを入れると、定期的にデスクトップ通知が届くようになります。

## ⚠️ セキュリティ上の注意
- この拡張機能は入力されたシークレットキーを `chrome.storage.local` に保存します。
- 第三者と共有するPCでの利用には十分注意してください。
- 重要なアカウントに使用する場合は、自己責任での利用をお願いします。
